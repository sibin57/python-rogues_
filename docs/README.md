# Выполнение практики
## Организация репозитория
Для начала потребовалось создать репозиторий и добавить всех участников в него как участников.

После этого потребовалось создать изначальный файл main.py с проектом в /src.

## Работа над проектом
Выполнение проекта проводилось в Visual studio с Python 3.14.5

Проект выполнялся паралелльно с гайдом расположенным по [ссылке](https://rogueliketutorials.com/).
